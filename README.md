# Email Classification System for Financial Services

This project implements an end-to-end machine learning pipeline for automatically classifying client emails into predefined business categories within a financial services environment.

Problem Statement

Financial institutions like RedRock process thousands of emails daily. Misclassification or delayed routing of emails can lead to regulatory risks, operational inefficiencies, and customer dissatisfaction.

This solution automates the classification process to ensure emails are accurately routed to the correct department.
